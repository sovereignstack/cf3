# Tread — your carbon footprint, simply

Tread is a **privacy-first, offline-first Progressive Web App** that helps one person understand,
track, and reduce their personal carbon footprint. It is **India-first** (₹, LPG, local trains,
two-wheelers, the Indian grid factor) and built to be calm, honest, and accessible.

- **No account, no server, no login.** All data lives in your browser (`localStorage`), under one key.
- **No network calls for your data, ever.** Zero third-party URLs. The only `fetch` is the service
  worker caching the app's *own* files for offline use. A reviewer can grep the source to verify it.
- **No build step, no dependencies.** The whole app is a single `index.html` (HTML + CSS + vanilla JS)
  plus a service worker, a manifest, and icons. Serve the static files as-is.
- **Light theme only**, with a high-contrast variant and full keyboard / screen-reader support.
- Figures are **estimates** from published average emission factors (see the in-app *About* and `PRD.md`).

## Run locally

A service worker needs HTTPS or `localhost` (not `file://`):

```bash
python3 -m http.server 8000      # then open http://localhost:8000
# or: npx serve
```

## Test

Dev-only tooling lives in [`dev/`](dev/) and never ships:

```bash
cd dev
npm install      # installs jsdom (dev-only)
npm test         # unit tests + headless smoke test (drives the real flow in jsdom)
npm run check    # quick syntax check of the inline script
```

CI runs the same tests on every push — see [`.github/workflows/ci.yml`](.github/workflows/ci.yml).

## Deploy

The app is just static files, so it deploys anywhere static:

- **GitHub Pages / any static host** — publish the repo root. All asset paths are relative, so it
  also works from a subfolder (e.g. a project-site URL).
- **Google Cloud Run** — a tiny static-file container ([`Dockerfile`](Dockerfile)) serves the same
  files on `$PORT`:

  ```bash
  gcloud run deploy tread --source . --region <region> --allow-unauthenticated
  ```

## Project layout

```
index.html             The whole app: HTML, CSS, vanilla JS.
manifest.webmanifest   PWA manifest (installable; maskable icons; theme).
sw.js                  Service worker: precaches the shell, serves cache-first, offline fallback.
icon-64/192/512.png    App icons (generated by dev/make_icons.py).
PRIVACY.md             Plain-language privacy policy (kept in sync with the in-app notice).
PRD.md                 Product requirements and roadmap.
CLAUDE.md              Working brief / hard constraints for contributors.
dev/                   Dev-only (never deployed): unit + smoke tests, package.json, icon generator.
```
